
# DriveEasy – Smart Vehicle Rental System

A modern **web-based vehicle rental application** built using **HTML5, CSS3, and JavaScript** that simulates a real-world vehicle rental platform.  
The system allows users to browse vehicles, view availability, and make reservations through an interactive and responsive interface.

---

#  Project Overview

DriveEasy is designed to demonstrate how **frontend web technologies** can be used to build a structured, responsive, and interactive web application.

The project focuses on:

- Clean webpage structure
- Responsive UI/UX
- Logical navigation flow
- Interactive client‑side features
- Organized project architecture

The system simulates a **vehicle rental service** where users can explore vehicles and reserve them easily.

# ⚙️ System Workflow

The workflow of the system follows a simple logical process.

### 1️⃣ User Opens the Website
The homepage displays the main interface with navigation and vehicle listings.

### 2️⃣ Vehicle Browsing
Users can browse available vehicles displayed in card format.

Each vehicle shows:

- Name
- Type
- Rental price
- Availability status

### 3️ Vehicle Selection
Users select a vehicle they want to rent.

### 4️⃣ Booking Process
The system processes the booking using JavaScript logic.

The vehicle status changes from:

Available → Reserved → Rented

### 5️⃣ UI Update
JavaScript dynamically updates the interface to reflect the new vehicle status.

### 6️⃣ Data Handling
Booking data is stored using **Local Storage** so the system retains information even after page refresh.
